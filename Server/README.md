## Installation

### Linux

```bash
python3 -m venv env
source env/bin/activate
pip install -r reqs.txt
```

### Bingdows

Chyba tak to sie robilo, sprawdzcie jak sie robi environment

```bash
./env/Scripts/Activate.sh
pip install -r reqs.txt
```
